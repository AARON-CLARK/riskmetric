By following [https://preview--condescending-lewin-a2fc51.netlify.com/packages/](R%20validation%20hub) validation information, riskmetric R package is created for proof of concept. All of content in this R package is draft only.

-   `data/riskmetric.Rd` contain the existing risk metrics generated by `riskmetric:::get_riskmetric` function.

Function Definition
===================

``` r

#> function () 
#> {

#>         `21CFR` = rep("N", 5), Codecov = c(83, NA, 88, 25, NA),

#>         Size = NA, External_language = NA)

#>             "Copyright", "Author", "BugReports", "Maintainer", 
#>             "Published", "Title", "URL")) %>% mutate(dependency = depends)

#> }
```

Load Data
=========

``` r
riskmetric <- readRDS("../data/riskmetric.Rd")
```
